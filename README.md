# 개발자 신원세 블로그

프론트엔드 개발자 신원세의 개인 블로그입니다.

## 기술 스택

### Frontend
- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Shadcn UI
- Radix UI

### CMS & Database
- Notion API
- Supabase

### Testing & Documentation
- Storybook
- Chromatic

### Infrastructure & Analytics
- Vercel
- Vercel Analytics
- Speed Insights

## 주요 기능

### 1. 블로그 포스트
- Notion을 CMS로 활용한 블로그 포스트 작성/관리
- Markdown/MDX 지원 (remark, rehype)
- 코드 하이라이팅
- 목차(TOC) 자동 생성
- 카테고리 태그 시스템

### 2. 프로젝트 쇼케이스
- 개인 프로젝트 포트폴리오 전시
- 프로젝트별 GitHub 및 서비스 링크 제공

### 3. 독서 기록
- 개발 관련 도서 리뷰
- 책 정보 및 커버 이미지 표시

### 4. 소개 페이지
- 경력 및 활동 이력 타임라인
